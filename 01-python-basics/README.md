# 🐍⚡ Lesson 01 — Python Basics for Traders

> Zero to useful. Fast.

## 🎯 Mission
Understand that Python is executable text and use it to automate tiny trading tasks.

## 🧠 Core ideas

- instructions run top to bottom
- variables store values
- lists and dictionaries store data
- `for` repeats work
- `if` adds decision logic
- functions package logic

## 📈 Trading angle

- store bids and asks
- compute spread and mid
- create orders
- process a small list of trades
- automate repetitive calculations

## 📦 Files

- `lesson.ipynb` — main class notebook
- `exercises/trading_system_exercises.ipynb` — 10-step build path
- `data/` — reserved for later
- `assets/` — optional visuals

## ✅ End state

By the end, the student should be able to:

- read simple Python
- change a value and rerun code
- write a small `for` loop
- define a simple function
- build a tiny trading-flavored system
