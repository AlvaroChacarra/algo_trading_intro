    Bloque 3 - De dict a objeto
    
      
    El mismo dato pasa por tres niveles: raw data, molde y ejemplar.
    
      
    El punto no es aprender una taxonomia. El punto es notar cuando un diccionario empieza a quedarse corto y una clase te deja juntar datos y comportamiento.
    

      
      
    
        
    ¿Por que el dict se queda corto?
    
        
    Intentas anadir comportamiento al dict — la operacion mas natural del mundo — y Python no puede.
    
        
    
          
    
            
    intentas esto
    
            
    
              
    
            
    
          
    
          
    
            
    Python te responde
    
            
    
              
    AttributeError: 'dict' object has
no attribute 'notional'

El dict guarda datos, pero no sabe
comportarse. La clase junta
datos + comportamiento bajo
una forma reusable.
    
            
    
          
    
        
    
      
    

      
    
        
    Cuando dict bastaSnapshots simples, prototipos rapidos y datos sin comportamiento.
    
        
    Cuando empieza a dolerRepites formulas, descripciones o validaciones fuera del dato.
    
        
    Que gana la classContrato visible, metodos junto al dato y menos dispersion mental.
    
      
    
      
    
        
    
          
        
    
        
    
          
    guia de lectura
